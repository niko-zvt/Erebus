# Wirth's Chainlet

A terribly stupid implementation of a singly linked list using prehistoric Pascal. There's plenty of room to break everything or shoot yourself in the foot.

### Build tools
- Free Pascal Compiler `fpc` or use any Online Pascal Compiler ([OnlineGDB](https://www.onlinegdb.com/online_pascal_compiler), [rextester](https://rextester.com/l/pascal_online_compiler), [codingground](https://www.tutorialspoint.com/compile_pascal_online.php))
